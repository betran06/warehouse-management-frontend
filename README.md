# Warehouse Management Frontend

Warehouse Management Frontend is a modern web dashboard built using React and Vite, designed to connect with the Warehouse Management Backend API.

This application provides role-based dashboards for Manager and Keeper to manage warehouse operations efficiently.

---

## 🚀 Features

- Secure Login Authentication
- Role-Based Dashboard (Manager & Keeper)
- Product & Category Display
- Warehouse Monitoring
- Merchant Data View
- Transaction Management
- API Integration with Laravel Backend
- Responsive UI Design

---

## 🛠️ Tech Stack

- React
- Vite
- JavaScript / TypeScript
- Axios / Fetch API
- Tailwind CSS (if used)

---

## 👥 User Roles

### 👨‍💼 Manager Dashboard
- Full system overview
- Manage products and categories
- Monitor warehouse and merchant data
- Manage transactions

### 👷 Keeper Dashboard
- View assigned merchant data
- View assigned product data
- Manage transactions

## 📦 Installation Guide

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/warehouse-management-frontend.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd warehouse-management-frontend
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run Development Server

```bash
npm run dev
```

---

## 🔗 API Configuration

Make sure backend API is running:

```
http://127.0.0.1:8000
```

Update API base URL in your configuration file:

```
VITE_API_URL=http://127.0.0.1:8000/api
```
